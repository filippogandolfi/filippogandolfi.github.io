After a few years of Hugo and Svelte experiments, this site is now built with **Astro**: the CV lives on the homepage, and this diary is where I'll capture experiences — projects, experiments, trips, and the occasional heroic save between the posts.

## How this works

Every post is a Markdown file. The frontmatter above this text defines the title, date, tags and cover photo. Photos dropped next to the posts get optimized automatically at build time, so I can upload iPhone shots without thinking about it.

> Fun fact: navigating this site makes little sounds. There's a toggle in the header if that's not your thing.

Publishing is one `git push` — GitHub Actions builds the site and GitHub Pages serves it. Total hosting cost: zero.
